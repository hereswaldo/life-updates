# Life Update:

A clickable prototype for a social app you open once a month, then close.

**Live demo:** https://hereswaldo.github.io/life-updates/

## Files

- `index.html` — landing page
- `prototype.html` — the app
- `assets/` — photos used in the demo

## What it does

Click around. Browse the feed. Try posting. Hit publish for the confetti. Tap the PRD button in the top bar to see open decisions.

## Status

Pre-build. Exploring the concept with friends. No build step. No dependencies. Just open `index.html`.
